#  Gestion de Notes des Étudiants

## membres

## Fonctionnalités
- Lecture d'un fichier CSV
- Calcul des moyennes de chaque étudiant
- Classement des étudiants par moyenne
- Sauvegarde des résultats dans resultats.txt

## Comment exécuter
1. Ouvrir le projet dans IntelliJ IDEA
2. Copier etudiants.csv à la racine du projet
3. Lancer Main.java
